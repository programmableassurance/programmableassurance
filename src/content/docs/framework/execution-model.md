# Execution Model

Governance executes whenever an organizational decision requires validation.

Examples include:

- infrastructure deployments
- identity changes
- financial approvals
- vendor onboarding
- AI model releases
- access requests

Each decision evaluates executable governance intent before an outcome occurs.

Execution is deterministic.

The same governance inputs should always produce the same governance decision.
